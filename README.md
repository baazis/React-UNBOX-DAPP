# React-UNBOX-DAPP
This is a react unbox dapp

## Working

mkdir react

cd react

truffle unbox react


truffle test

truffle compile


Test dapp:            cd client && npm test

Run dev server:       cd client && npm run start

Build for production: cd client && npm run build


“npm run start” will fire up your React.js site.
